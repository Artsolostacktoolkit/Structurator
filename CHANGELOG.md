# История изменений StructureSnap

Все существенные изменения в этом проекте будут задокументированы в этом файле.

## [1.0.0] - 2026-03-29

###  Добавлено
- **Экспорт структуры проекта** в 4 формата:
  - JSON (машиночитаемый формат)
  - PNG (визуализация дерева)
  - Tree (текстовое дерево)
  - CSV (табличный формат)
  
- **Парсинг решений Visual Studio**:
  - Поддержка форматов .sln и .slnx
  - Использование MSBuild Locator для точного анализа
  - Асинхронная загрузка с прогресс-баром
  - Отмена длительных операций

- **Сбор кода для нейросетей**:
  - Экспорт всех файлов проекта в один Markdown файл
  - Манифест с оглавлением
  - Подсветка синтаксиса для разных языков
  - Подсказки о роли файлов (ViewModel, Service, Controller)

- **Селективная очистка комментариев**:
  - Удаление обычных комментариев
  - Сохранение XML-документации (/// <summary>)
  - Сохранение служебных меток (TODO, FIXME, NOTE)
  - Исключение файлов БД из очистки (.sql, .db, .edmx)
  - Статистика экономии (строки, байты, проценты)

- **Современный UI**:
  - Тёмная тема с зелёными акцентами (#4CAF50)
  - Карточки форматов с превью
  - Плавные анимации появления
  - Адаптивный дизайн (MinWidth: 1100px)
  - Перетаскивание окна без заголовка

- **MVVM архитектура**:
  - Разделение логики и представления
  - Dependency Injection через конструктор
  - RelayCommand для команд
  - INotifyPropertyChanged для привязок

###  Технические улучшения
- Миграция на .NET 10.0-windows
- Оптимизация чтения файлов с автоопределением кодировки
- Параллельная обработка с SemaphoreSlim (макс. 4 потока)
- Очистка временных файлов PNG при закрытии
- Логирование в Debug Console

###  Исправлено
- Прогресс-бар теперь работает асинхронно
- Кнопка "Выбрать решение" блокируется после загрузки
- Сброс данных очищает превью карточек
- Корректная обработка путей с кириллицей
- Фильтрация системных элементов MSBuild

###  Статистика
- Всего файлов: 24
- Всего строк кода: ~4,200
- Поддерживаемых расширений: 40+
- Исключаемых папок: 7

###  Безопасность
- Валидация путей и расширений
- Ограничение размера файлов (10 MB)
- Исключение чувствительных папок (bin, obj, .git)
- Параметризованная работа с файловой системой

###  Документация
- Подробные XML-комментарии в коде
- README с примерами использования
- CHANGELOG с историей изменений
